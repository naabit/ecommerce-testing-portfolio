# E-commerce Testing Portfolio

Proyecto de testing funcional y automatización sobre una aplicación e-commerce desarrollada en Django.

## Objetivo

Validar los principales flujos de usuario, identificar errores y aplicar buenas prácticas de testing manual y automatizado.

## Alcance

Se probarán los siguientes módulos:

- Registro de usuario
- Inicio de sesión
- Catálogo de productos
- Detalle de producto
- Carrito de compras
- Proceso de compra (checkout)

## Tipos de testing aplicados

- Testing funcional manual
- Testing exploratorio
- Testing de regresión (básico)
- Automatización de flujos críticos (Cypress o Playwright)

## Herramientas

- Documentación: Markdown
- Automatización: (por definir)
- API Testing: Postman (segun necesidad de alcance)
